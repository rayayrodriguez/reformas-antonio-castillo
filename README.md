# Reformas Antonio Castillo - Web Oficial

Página web profesional para **Reformas Antonio Castillo**, empresa de reformas y construcción en Jaén.

## 🌐 Ver la web

Una vez activado GitHub Pages, la web estará disponible en:
**https://rayayrodriguez.github.io/reformas-antonio-castillo/**

## 📋 Cómo activar GitHub Pages (una sola vez)

1. Ve a: https://github.com/rayayrodriguez/reformas-antonio-castillo/settings/pages
2. En "Source", selecciona **"Deploy from a branch"**
3. En "Branch", selecciona **"master"** y **"/ (root)"**
4. Haz clic en **"Save"**
5. Espera 1-2 minutos y la web estará online

## 📸 Cómo añadir fotos reales

Para añadir las fotos reales de los trabajos de Antonio:
1. Descarga el repositorio o edita directamente en GitHub
2. Sube las fotos a la carpeta `images/`
3. Edita `index.html` y cambia las rutas de las imágenes
